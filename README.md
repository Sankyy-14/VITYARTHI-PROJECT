# Student Academic Performance Analyzer

A Python-based tool to analyze student marks, evaluate performance trends,
identify weak subjects, and generate insights with visualizations.

## Features
- Load and clean CSV marks data
- Calculate total, percentage, grade
- Subject-wise and class-wise analytics
- Generate charts and performance reports

## Technologies
- Python 3.8+
- pandas
- matplotlib
- numpy

## How to Run
```bash
pip install -r requirements.txt
python src/main.py
```

## Output
- Terminal insights
- Bar/line charts (matplotlib)
- Markdown/HTML report (report.md). Convert to PDF using your favorite tool.
